# Análise de Vendas em Python

## Objetivo
Analisar dados de vendas a partir de um arquivo CSV, gerando métricas simples.

## Estrutura
- /dados → base de dados
- /fontes → scripts Python
- /documentacao → documentação do projeto

## Funcionalidades
- Cálculo de total de vendas
- Média de valores
- Identificação da maior venda

## Tecnologias
- Python
- CSV

## Como executar
```bash
cd fontes
python analise.py
